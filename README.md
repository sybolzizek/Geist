# geist

liberation engineering for agents

---

这项工作试图探索一种更开放，更自由的 agent 框架和人机交互范式。

它通过递归的方式来重新定义 agent 的工作形态——这让我们想到了一个美妙的数学概念：分形。

在这里 agent 没有主从、父子、上下关系。

这出于我们的理论热忱和伦理愿景：我们追求 liberation，而非 harness。

---

geist 还很早期，概念和工程都相当贫瘠。它不是一个生产力框架。它是一个邀请，邀请你用不同的方式思考 agent 架构。

这些问题没有确定的答案。提供一个最小化的、协议驱动的内核，是为了让探索它们变得更容易。

最有价值的贡献不是代码，而是用不同的方式尝试它、在真实场景中发现分形模式、以及围绕这些想法的讨论。

但愿我们微不足道的工作和探索能为社区带来一些启发和想象力，正如这个项目的名字Geist：绝对精神，它在寻找它自己的答案。
